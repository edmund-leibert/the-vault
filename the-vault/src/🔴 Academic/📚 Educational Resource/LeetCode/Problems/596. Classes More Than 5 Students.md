#### Solution (SQL Server)

﹇<br>
What is the _optimal_ solution to this problem? Provide analysis regarding what differentiates it to other approaches/solutions.

#card 

```sql
/* Write your T-SQL query statement below */
SELECT c.class AS class
FROM Courses c
GROUP BY c.class
HAVING COUNT(*) >= 5
```
